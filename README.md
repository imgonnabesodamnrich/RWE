## 研究背景与数据增强的必要性

在基于深度学习的智能故障诊断领域，模型的性能通常依赖于大量且类别平衡的训练数据。然而，在航空液压泵等高可靠性设备的实际运维中，存在显著的“数据匮乏”与“类别不平衡”问题：

1.  **故障样本稀缺**：关键设备发生故障的概率极低，且部分故障数据在工程现场未能被有效保存。
2.  **模型偏见**：由于健康样本数量远多于故障样本，直接使用原始数据训练会导致模型倾向于将所有样本预测为“健康”，从而失去诊断能力。
3.  **现有方法的局限**：传统的信号分析方法依赖专家经验，而生成对抗网络（GAN）等数据驱动方法在振动信号生成上存在训练不稳定及物理意义缺失的问题。

针对上述问题，本文提出了一种基于信号处理机理的数据增强方法 RWE，旨在通过小样本生成高质量的合成故障信号。

## 核心方法：随机小波扩展 (RWE) 原理

RWE 是一种专门针对一维时间序列（振动信号）设计的增强策略。其核心逻辑在于利用小波变换对信号的时频特征进行微调，生成既保留原始故障特征，又具有统计多样性的新样本。

### 设计思路

机械故障的特征主要体现在**频率成分**上。为了确保生成的样本有效，方法设计遵循以下原则：
*   **特征保留**：保持信号的主要频率成分不变，确保合成样本属于同一故障类别。
*   **引入多样性**：通过随机放大特定频段的小波系数，模拟同一故障类别下的不同表现形式。
*   **统计一致性**：强制合成样本具有与原始样本相同的均值和标准差，避免引入无关的能量偏差。

### 算法实施步骤

RWE 的具体流程如下（参考原论文 Fig. 4）：

1.  **零均值化**：将原始振动信号减去其平均值。
2.  **统计量记录**：计算原始信号的标准差 ($\sigma_1$)。
3.  **小波包分解 (WPD)**：利用小波包变换将信号分解为不同频段的系数（例如使用 DB4 小波进行 3 层分解）。
4.  **随机扩展 (Random Expansion)**：
    *   随机选择一个小波包节点（代表特定频段）。
    *   对该频段的系数幅度进行非线性放大。放大公式为 $y = \text{sign}(x) \cdot (\text{abs}(x) + \tau)$，其中 $\tau$ 为随机扩展量。
5.  **小波包重构 (WPR)**：利用修改后的系数和其余未修改的系数重构回时间序列信号。
6.  **再次零均值化**：对重构信号去均值。
7.  **标准化变换**：计算重构信号的标准差 ($\sigma_2$)，并将信号按比例缩放 $t = s \cdot (\sigma_1 / \sigma_2)$，使其标准差回归到原始水平 $\sigma_1$。

通过该流程，生成了一组在频域能量分布上有细微差异，但核心故障模式与原信号一致的扩充样本。

## 实验验证与结果分析

为验证 RWE 的有效性，研究在以下三个数据集上进行了测试，并结合一维卷积神经网络 (1D-CNN) 进行了小样本分类实验：

1.  **航空液压泵数据集**（本文自建）：包含健康状态及5种典型故障（如滑靴磨损、配流盘磨损等）。
2.  **凯斯西储大学 (CWRU) 轴承数据集**：公开基准数据集。
3.  **齿轮箱数据集**：来自传动系统模拟器。

**对比实验设置**：
将 RWE + CNN 与以下方法进行了对比：
*   **Baseline CNN**：仅使用原始少量样本训练。
*   **DS + CNN**：下采样（减少健康样本）。
*   **OS + CNN**：简单过采样（直接复制故障样本）。
*   **SMOTE / ADASYN**：传统的插值类过采样算法。

**实验结论**：
在小样本设置下（例如每类故障仅有 5 个训练样本），**RWE + CNN** 在 F1 分数、精确率（Precision）和召回率（Recall）等指标上均优于对比方法。这表明 RWE 生成的样本比单纯的复制或线性插值更具有特征代表性，能有效提升模型对未见样本的泛化能力。

## 结论

本文提出的 RWE 方法具有以下特点：
1.  **轻量级与稳定性**：基于确定性的信号处理理论，无需训练生成器，计算成本低且结果可控。
2.  **物理可解释性**：通过改变特定频段的小波系数幅度来生成样本，符合机械振动信号的物理特性。
3.  **适用性强**：实验证明该方法能有效缓解小样本导致的过拟合问题，适用于航空液压泵及其他旋转机械的故障诊断任务。
